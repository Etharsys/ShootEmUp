======================
HOW TO LAUNCH THE GAME
======================

Double tap on the file named : 'ShootEmUp.sh', then select launch.
Or go on a terminal and lauch the same file as an executable : '$ ./ShootEmUp.sh'

===
LIB
===

ShootEmUp use a grahical library from Université Gustave Eiffel (ex : Université Paris-Est-Marne-La-Vallée).
This library is used as a local library on the bin folder (don't delete it).
